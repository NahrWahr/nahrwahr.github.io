# Orbital Simulator & GNC Systems

Designed an **Orbital simulation software stack** on par with STK (commercial SOTA software) for high-fidelity satellite orbit simulation. 

## Technical Overview
- **GNC Systems:** Implemented Guidance, Navigation, and Control systems specifically for satellite rendezvous.
- **Control Strategy:** Utilized **Non-linear Model Predictive Control (NMPC)** to handle complex docking maneuvers and orbital constraints.
- **Simulation Fidelity:** Engineered to match industry-standard benchmarks for orbit propagation and state estimation.
